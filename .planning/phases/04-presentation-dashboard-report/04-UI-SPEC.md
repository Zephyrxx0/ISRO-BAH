## Spacing Scale

## Layout Contract

### Home Page (`/`)

```
┌─────────────────────────────────────────────────────┐
│ Navbar: [Candidates] [Star Map] [About]             │
├─────────────────────────────────────────────────────┤
│ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐               │
│ │Total │ │Gold  │ │Planets│ │AvgSDE│  ← Stat cards │
│ └──────┘ └──────┘ └──────┘ └──────┘               │
├─────────────────────────────────────────────────────┤
│ [Tier ▼] [Class ▼] [Period: min–max]  ← Filter bar │
├─────────────────────────────────────────────────────┤
│ TIC ID │ Period │ Depth │ SDE │ Class │ Tier │  →   │
│ ─────────────────────────────────────────────────── │
│ row    │ row    │ row   │ row │ row   │ row  │[CTA] │
│ row    │ row    │ row   │ row │ row   │ row  │[CTA] │
│ ...                                                 │
├─────────────────────────────────────────────────────┤
│ Pagination: ← 1 2 3 ... →                          │
└─────────────────────────────────────────────────────┘
```

Focal point: Candidate table with "Explore Candidate" CTA per row.

### Star Detail Page (`/star/[ticid]`)

```
┌─────────────────────────────────────────────────────┐
│ Navbar                                              │
│ Breadcrumb: Home > TIC 12345678                     │
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐ │
│ │ Parameter Card: P, δ, T₁₄, SDE, SNR, Rp/Rs,   │ │
│ │ inclination ± uncertainties   [Gold Badge]      │ │
│ └─────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────┤
│ 4-Panel Diagnostics (2×2 grid):                     │
│ ┌──────────────┐ ┌──────────────┐                   │
│ │ Raw+Detrend  │ │ Periodogram  │                   │
│ └──────────────┘ └──────────────┘                   │
│ ┌──────────────┐ ┌──────────────┐                   │
│ │ Phase-folded │ │ Softmax bars │                   │
│ └──────────────┘ └──────────────┘                   │
├─────────────────────────────────────────────────────┤
│ MCMC Corner Plot (if available)                     │
├─────────────────────────────────────────────────────┤
│ TRICERATOPS+ FPP/NFPP + Disposition Badge           │
└─────────────────────────────────────────────────────┘
```

Focal point: Parameter card with disposition badge.

### Star Map Page (`/map`)

```
┌─────────────────────────────────────────────────────┐
│ Navbar                                              │
├─────────────────────────────────────────────────────┤
│ [✓Gold] [✓Silver] [✓Bronze]  ← Tier filter toggles │
├─────────────────────────────────────────────────────┤
│                                                     │
│         Leaflet.js Equirectangular Map              │
│         RA (0-360°) × Dec (-90° to +90°)           │
│         Dark background, no external tiles          │
│         Markers: ● Gold  ● Silver  ● Bronze        │
│         Radius scaled by SDE                        │
│         Hover: TIC ID, period, depth, class         │
│         Click: navigate to /star/[ticid]            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

Focal point: Map with Gold markers prominent (largest radius + brightest color).
