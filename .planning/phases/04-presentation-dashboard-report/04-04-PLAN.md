  </read_first>
  <action>
1. Install packages: `cd dashboard && pnpm add leaflet react-leaflet && pnpm add -D @types/leaflet`

2. Create dashboard/components/star-map.tsx:
   - 'use client' directive (REQUIRED — Leaflet needs window/document)
   - Import leaflet CSS: `import 'leaflet/dist/leaflet.css'`
   - Import from react-leaflet: MapContainer, CircleMarker, Tooltip
   - Import L from 'leaflet' for CRS.Simple and latLngBounds
   - Import useRouter from 'next/navigation'

3. Define Star type: { tic_id: string, ra: number, dec: number, sde: number, tier: 'Gold'|'Silver'|'Bronze', period: number, depth: number, classification: string }

4. Define tierColor map: { Gold: '#f59e0b', Silver: '#94a3b8', Bronze: '#d97706' }

5. Component props: { stars: Star[] }

6. State: visibleTiers = useState<Set<string>>(new Set(['Gold', 'Silver', 'Bronze']))

7. Filter logic: `const filteredStars = stars.filter(s => visibleTiers.has(s.tier))`

8. Tier toggle UI above map:
   - 3 checkbox+label pairs: [✓] Gold [✓] Silver [✓] Bronze
   - Checkbox toggles the tier in visibleTiers Set
   - Label color matches tier color

9. MapContainer config per D-12:
   - crs={L.CRS.Simple}
   - bounds={L.latLngBounds([[-90, 0], [90, 360]])}
   - maxBounds same as bounds, maxBoundsViscosity=1.0
   - minZoom={-1}, maxZoom={4}, zoomSnap={0.5}
   - style={{ height: '100%', width: '100%', background: '#0f172a' }}
   - NO TileLayer — dark background is the map itself

10. CircleMarker for each filtered star:
    - center={[star.dec, star.ra]} (Leaflet: [lat, lng] = [dec, ra])
    - radius={Math.max(4, star.sde / 2)} — scaled by SDE per D-13
    - pathOptions: color and fillColor from tierColor[star.tier], fillOpacity=0.8, weight=1
    - eventHandlers={{ click: () => router.push(`/star/${star.tic_id}`) }}
    - Tooltip content: "TIC {tic_id}\nP={period}d, δ={depth}ppm\n{classification}"
  </action>
