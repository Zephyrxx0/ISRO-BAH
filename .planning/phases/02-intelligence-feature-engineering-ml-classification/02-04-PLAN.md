# Plan 4: CNN Fine-Tuning + XGBoost Training

<objective>
Create the two standalone training scripts that produce the classification models during the hackathon. train_cnn_finetune.py loads kepler_pretrained.h5, freezes early conv layers, and fine-tunes on ExoFOP-TESS labels with 7× augmentation via TransitDataGenerator. train_xgboost.py trains a 4-class XGBoost classifier on the 8 engineered features with stratified k-fold CV, SHAP TreeExplainer feature importance, and full evaluation metrics (confusion matrix, per-class F1, ROC-AUC). Both scripts log everything to MLflow under the 'tess-finetune-xgboost' experiment.
</objective>

<action>
Create `src/phase2/train_cnn_finetune.py` as a standalone training script with:

1. Imports: `os`, `argparse`, `subprocess`, `numpy`, `pandas`, `tensorflow` (mixed_precision, callbacks), `sklearn.model_selection.train_test_split`, `sklearn.utils.class_weight.compute_class_weight`, `mlflow`, `src.phase2.data_generator.TransitDataGenerator`

2. Constants: `SEED = 42`, `MODEL_INPUT_PATH = 'data/models/kepler_pretrained.h5'`, `MODEL_OUTPUT_PATH = 'data/models/cnn_finetuned.h5'`

3. `get_git_hash()` helper (same as train_kepler.py)

4. `def main(args):` function that:
   a. Calls `tf.keras.backend.clear_session()` and sets `mixed_float16` policy
   b. Loads `kepler_pretrained.h5` with `compile=False`
   c. Freezes first 12 layers (first 3 Conv1D+MaxPool pairs in each tower = early feature detectors)
   d. Compiles with Adam(lr=args.lr), sparse_categorical_crossentropy, sparse_categorical_accuracy
   e. Loads master Parquet, filters to rows where `label` column is not NaN (ExoFOP-labeled candidates)
   f. Performs 3-way stratified split: 60% train, 20% calibration, 20% test using `train_test_split` with `random_state=SEED`
   g. Verifies split integrity: no TIC ID overlap between splits (assertion)
   h. Creates `TransitDataGenerator(train_df, batch_size=args.batch_size, augment=True)` and `TransitDataGenerator(val_df, batch_size=args.batch_size, augment=False)`
   i. Computes class_weight via `compute_class_weight('balanced', ...)`
   j. Trains with callbacks: ModelCheckpoint (save_best_only, monitor val_sparse_categorical_accuracy, mode='max'), EarlyStopping(patience=10), ReduceLROnPlateau(factor=0.5, patience=5)
   k. Logs to MLflow experiment 'tess-finetune-xgboost', run_name 'cnn-finetune'
   l. Saves calibration and test split indices to `data/models/split_indices.npz` for downstream use

5. `if __name__ == '__main__':` with argparse: `--epochs` (default 50), `--batch-size` (default 32), `--lr` (default 1e-4), `--catalogue` (default 'data/catalogue/master.parquet')

Key code for layer freezing:
```python
base_model = tf.keras.models.load_model(MODEL_INPUT_PATH, compile=False)
# Freeze first 12 layers (3 Conv+Pool pairs per tower × 2 towers)
for layer in base_model.layers[:12]:
    layer.trainable = False
```

Key code for 3-way split:
```python
train_df, temp_df = train_test_split(labeled_df, test_size=0.4, stratify=labeled_df['label'], random_state=SEED)
cal_df, test_df = train_test_split(temp_df, test_size=0.5, stratify=temp_df['label'], random_state=SEED)
```

<action>
Create `src/phase2/train_xgboost.py` as a standalone training script with:

1. Imports: `os`, `argparse`, `subprocess`, `numpy`, `pandas`, `xgboost`, `sklearn.model_selection` (StratifiedKFold, train_test_split), `sklearn.metrics` (f1_score, classification_report, confusion_matrix, roc_auc_score, ConfusionMatrixDisplay), `sklearn.utils.class_weight.compute_class_weight`, `shap`, `matplotlib.pyplot`, `mlflow`

2. Constants:
```python
SEED = 42
MODEL_OUTPUT_PATH = 'data/models/xgboost_ensemble.json'
FEATURE_COLUMNS = [
    'odd_even_depth_diff', 'secondary_eclipse_depth', 'centroid_shift_sigma',
    'v_shape_metric', 'crowdsap', 'duration_period_ratio', 'tls_sde', 'tls_snr',
]
CLASS_NAMES = ['PC', 'EB', 'Blend', 'StellarVar']
```

3. `def main(args):` function that:
   a. Loads master Parquet, filters to labeled rows (`label` not NaN)
   b. Loads split indices from `data/models/split_indices.npz` (same split as CNN to ensure fair comparison)
   c. Extracts feature matrix `X = df[FEATURE_COLUMNS].values` and labels `y = df['label'].values.astype(int)`
   d. Runs stratified 5-fold CV on training set to report mean±std F1-macro
   e. Trains final XGBoost model on full training set with these params:
     ```python
     params = {
         'objective': 'multi:softprob',
         'num_class': 4,
         'tree_method': 'hist',
         'device': 'cuda',
         'max_depth': 6,
         'learning_rate': 0.05,
         'n_estimators': 500,
         'subsample': 0.8,
         'colsample_bytree': 0.8,
         'min_child_weight': 3,
         'reg_lambda': 1.0,
         'reg_alpha': 0.1,
         'eval_metric': 'mlogloss',
         'early_stopping_rounds': 20,
         'random_state': SEED,
     }
     ```
   f. Evaluates on test set: accuracy, per-class F1, confusion matrix, ROC-AUC (one-vs-rest)
   g. Runs `shap.TreeExplainer(clf).shap_values(X_test)` and saves summary bar plot to `outputs/shap_summary.png` (150 dpi)
   h. Saves model to `data/models/xgboost_ensemble.json`
   i. Logs all params, metrics, confusion matrix PNG, SHAP plot, and model artifact to MLflow experiment 'tess-finetune-xgboost', run_name 'xgboost-4class'

4. `if __name__ == '__main__':` with argparse: `--catalogue` (default 'data/catalogue/master.parquet'), `--n-folds` (default 5)

Key code for SHAP:
```python
explainer = shap.TreeExplainer(clf)
shap_values = explainer.shap_values(X_test)
shap.summary_plot(
    shap_values, X_test, feature_names=FEATURE_COLUMNS,
    class_names=CLASS_NAMES, plot_type='bar', show=False,
)
plt.savefig('outputs/shap_summary.png', dpi=150, bbox_inches='tight')
plt.close()
```
